# 2026-07-17 固定コメント未観測範囲の検証計画

## 目的

CO v4.1.0までに未観測だった固定コメントの配置・重ね・overflow規則を、保存済み公式バンドルの隔離再生で境界値まで測定する。個別CAの検出や作品固有条件を使わず、通常コメントとコメントアートへ共通に適用できる一般規則を導出する。

## 完了条件

1. 公式バンドルで固定コメントの候補位置探索とfallback条件を説明できる。
2. 時刻差、位置、size、行数、`full`、font、`ender`、処理順、viewportの各分岐を最低1つの反証ケースで覆う。
3. 導出に使っていない実CA全11バッチをホールドアウトとして再現する。
4. COの単体テストで境界値と通常固定コメントの非退行を固定する。
5. 実装後の合成プローブで公式とCOの座標・描画順が一致する。

## 実験軸

### 時間

固定コメントの表示時間内と境界外を分離する。

- 同一vpos
- `+1, +10, +33, +100, +250, +500, +1000, +2000, +2999, +3000, +3001, +5000ms`

### 縦寸法

単行、複数行、自動縮小の直前・開始点、表示高付近、表示高超過を使う。

| size   | 行数            |
| ------ | --------------- |
| small  | 1, 6, 7, 16, 30 |
| medium | 1, 4, 5, 16, 30 |
| big    | 1, 2, 3, 16, 30 |

### コマンド

- 位置: `ue`, `shita`
- `full`: あり / なし
- font: `defont`, `gothic`, `mincho`
- `ender`: あり / なし
- 色: white / black / red。色が幾何へ影響しない対照として使用する

### 順序

- 配列順と `no` がともに昇順
- 配列順だけ逆順
- `no` だけ逆順
- 同一vposの3〜5レイヤー

### viewport

- 1280×720
- 1600×900
- 960×540
- 720×1280

### 位置間相互作用

- `ue` 同士
- `shita` 同士
- `ue` と `shita` の同時投入
- 通常 `naka` 予約との同時投入

## 判定値

各コメントについて次を保存する。

- outer Canvas上のtranslation X/Y
- source canvas幅・高さ
- 処理sequence
- 同一座標へ重なったか
- 既存コメントの次へ積まれたか
- 表示対象から外れたか
- run間で座標が変動したか

同一条件を最低3回反復し、fallbackが決定的かランダムかを分離する。

## ホールドアウト

`so31723295` 第5話EDの11バッチ、計48コメントは一般規則の導出には使わず、最終検証にのみ使う。各バッチは同一vposの3〜5コメント、16行、`ue big full mincho`、`ender`なしで、矩形・白抜き・歌詞を別レイヤーとして持つ。

## 実装禁止事項

- `ca` コマンド、作品ID、コメント番号範囲、本文文字種で特殊分岐しない。
- 「同一vposなら常に重ねる」と観測前に決めない。
- 公式の座標境界を測らず、COの現在値へ合わせて教師を解釈しない。
- 通常固定コメントの衝突回避を破壊する変更を、ホールドアウトだけの成功で採用しない。
