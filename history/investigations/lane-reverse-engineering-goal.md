# ニコニココメントレーン再現の次目標

> 旧評価基盤から見た当時の目標です。後続の824ケース隔離実験により置き換えられています。

目的:
- ニコニコ動画の通常コメントレーン選択をリバースエンジニアリングし、`comment-overlay` の `naka` レーン配置を公式プレイヤーへ機械的に近づける。
- 小さな定数調整ではなく、公式プレイヤーが実際に描画したコメント集合・描画時刻・描画Y・レーン占有判定をCO内部の `laneDecision` と1対1で比較できる評価基盤を作る。

現状の距離:
- 通常コメント9プローブの合成スコアは `61.9%`。
- 平均絶対レーン差は約 `1.85` レーンで、まだ再現とは言えない。
- 低スコア区間では `teacher 60 / candidate 19` のように、レーン選択以前に教師とCOのコメント集合が一致していないケースが残る。

足りないもの:
- 公式プレイヤーが実際に描画したコメントID集合の復元。
- 公式traceの `drawImage` / source canvas / text metadata を同一コメント単位に紐付ける処理。
- COへ公式と同じコメント集合だけを流す再生入力。
- 公式の描画Y・font size・sourceHeight・vposから、実レーンYモデルを推定する処理。
- CO内部 `laneDecision` と公式側の同一コメント描画結果を直接比較する評価器。
- 予約矩形、追い越し条件、発火時刻補正、レーン再利用条件を分解できる診断出力。

次にやること:
- CDP教師traceからコメント単位の `text/vpos/sourceCanvas/drawImage y/x/size/time` を復元する。
- CO候補traceにも同じコメント単位キーを出し、教師と候補をコメント単位で対応付ける。
- 既存の軌跡単位スコアを補助指標に下げ、主評価を「同一コメントのlane/y/time一致」に移す。
- コメントアート区間は通常 `naka` レーン位置取りプローブから外し、別の描画プリミティブ評価として扱う。
