# 変更履歴

このプロジェクトの重要な変更はこのファイルに記録します。書式は [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) に基づきます。

## [Unreleased]

### Added

- 公式サイトへの反復アクセスなしでコメント互換性を研究できるように、匿名セッションの記録、SHA-256検証、外部通信を遮断した再生、通信監査を行う研究基盤を追加した。
- 研究ツール、研究計画、ローカル成果物の配置を一貫させるため、ニコニココメントシステム研究の入口を `research/` に集約した。
- 遅延ロードされるworkerを過剰な再キャプチャなしで補えるように、公式ホストと静的拡張子だけを許可するアーカイブ追補ツールを追加した。
- 公式描画コードの一般規則を合成入力から測定できるように、`nvComment` 応答のシナリオ置換、video時刻のseek、Canvas描画観測、レーン解析、同幅2コメントの時刻差プローブを研究基盤へ追加した。
- 公式プレイヤーのコマンド挙動を非公式フィルターと混同しないように、NicoCacheが挿入した `/local/` 資産を隔離再生から除外して個別監査する処理を追加した。
- サイズ・行数が異なるコメントの一般規則を分離できるように、合成コメントプローブへコメントごとのsize・行数と自動生成本文の長さを独立指定する機能を追加した。
- 表示高を使い切った後の規則を反証できるように、同時14コメントの独立再生で非量子化ランダムYとrun間変動を確認するoverflowシナリオを追加した。
- コメントアートの視覚退行を確認できるように、『ようこそ実力至上主義の教室へ』第5話のED区間と、旧leafを含む公式mainコメントを `overlay-tests` presetへ追加した。

### Changed

- `overlay-tests/fixtures` の `*-comments.json` と同名動画の組をビルド時に検出し、動画選択プルダウンをコメントデータに応じて自動生成するようにした。

- サイズや複数行が混在しても公式プレイヤーと同じ縦配置になるように、通常コメントの固定レーンを768基準の文字寸法、行数別自動縮小、可変高区間の上優先予約へ置き換えた。
- 公式Canvasの論理解像度と表示寸法を混同しないように、通常コメントのテクスチャ余白と描画位置を実描画文字幅基準へ変更した。
- 固定コメントの縦配置を公式プレイヤーの境界規則へ合わせた。単体が表示高未満なら空き区間へ整列し、空き不足時だけランダム位置へ退避する。単体が表示高以上なら後続レイヤーも同じ上端または下端へ重ね、コメントアートの矩形・白抜き・歌詞レイヤーを保持する。
- 文字種や長さに依存せず固定コメントの占有高を再現できるように、実測文字幅、`full` / `ender`、複数行の縮小順序、最小フォント後の0.1刻みscaleを一般化した幅フィッターへ置き換えた。
- 横流れコメントを特定CAへ過適合させず再現できるように、速度、開始・退出、同一レーン再利用を実測文字幅と4秒横断式へ統一し、`full`や同一vposの特定コマンドだけを変形する分岐を削除した。
